# **Getting Started**

## **Prerequisites**

Make sure you have:

* **Programming Language:** TypeScript
* **Package Manager:** npm

---

## **Installation**

### 1. Clone the repository:

```sh
git clone https://github.com/Nx-One/portfolio
```

### 2. Navigate to the project directory:

```sh
cd portfolio
```

### 3. Install the dependencies using npm:

```sh
npm install
```

---

## **Usage**

Start the development server:

```sh
npm start
```

---
